# Neural Networks (Elsevier)

## Overview

## Submission Requirements
- Length: 12-15 pages
- Double-blind review: No (single-blind)
- Code availability: Strongly encouraged
- Supplementary material: Permitted

## Fit Assessment

- **Moderate fit**: Application-focused work without architectural novelty
- **Weak fit**: Pure systems or database work without neural component
