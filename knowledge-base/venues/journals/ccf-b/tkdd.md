# ACM Transactions on Knowledge Discovery from Data (TKDD)

## Overview
ACM TKDD is the premier CCF-B journal for data mining and knowledge discovery research. It regularly publishes work on graph mining, anomaly detection in dynamic networks, and graph neural network methods. The 2024 comprehensive survey on dynamic graph anomaly detection (Ekle & Eberle, 44 pages) was published here, establishing it as a primary venue for this sub-area.

## Submission Requirements
- Length: 12-15 pages (ACM format)
- Double-blind review: Yes
- Code availability: Expected but not mandatory
- Supplementary material: Permitted

## Fit Assessment
- **Strong fit**: Graph neural network methods, dynamic graph analysis, anomaly detection, graph representation learning
- **Moderate fit**: Pure deep learning without graph structure, methods evaluated only on synthetic data
- **Weak fit**: Pure theory without empirical validation, vision or language tasks without graph component

## Typical Acceptance Timeline
- First decision: 3-6 months
- Revision cycles: 1-2 rounds typical
- Total time to publication: 6-12 months
