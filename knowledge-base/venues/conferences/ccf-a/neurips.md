# Neural Information Processing Systems (NeurIPS)

## Overview
CCF-A conference (~25% acceptance). Top venue for machine learning research. Strong preference for theoretical novelty or fundamental methodological insights. For GNN work: needs formal analysis (expressiveness, convergence, generalization) beyond empirical results. For VLA/VLM: novel training paradigms, representation learning advances.
